# JS
```
$(clickMe).on('click',function(){
  $(popover).show()
  setTimeout(function(){
    $(document).one('click',function(){
      $(popover).hide()
    })
  },0)
})

$(wrapper).on('click', function(e){
  e.stopPropagation()
})

// 加一个 setTimeout，是为了防止点击 button 的时候立即触发了
// 把浮层 hide() 的动作。有了 setTimeout ，作用就是把
// 点击出现浮层 和 hide 副词两个动作隔开一点。



// html 里加一个 checkbox 就要加一些代码。如上。在 wrapper 上
// 加 stopPropagation() 阻止事件冒泡，不传播到 document 那
// 一层，于是就不会触发 document 的 hide() 代码。bug 就解决了。
```
