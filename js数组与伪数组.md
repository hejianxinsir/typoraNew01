4. 数组的长度
```

// 只有一个参数的时候，这个参数就是数组的长度

var b = Array(3,3)
// 有两个参数的时候，这两个就是数组的内容了，所以该数组的长度 b.length === 2

这种区别叫做「不一致性」

var a = new Array(3,3)  // 加不加 new 对 Array 来说都一样。
```

for(let i=0; i<a.length; i++){
    console.log(a[i])

20. reduce 
```
以下是用 reduce 求和
a = [1,2,3,4,5]

a.reduce( function(sum,n){
    return sum + n 
},0 )
// 15

// 还可以用箭头函数写：a.reduce( (sum,n) => sum + n , 0  )  一样的效果

reduce 可以用来求和。reduce 括号里面必须有两个参数，一个是函数，另一个是初始值。函数里必须有两个参数，第一个是上一次的结果，第二个是数组里的某个数值 n 。 意思是初始的总值为 0 ，然后把上一次的结果再加数组里的某个值，一直加完，就是总合计的值了。


// 另一个例子：对所有奇数求和
var a = [1,2,3,4,5,6,7,8,9]

a.reduce( function(arr,n){
	if( n%2 !== 0 ){
		arr.push(n)
	}
	return arr 
},[] ).reduce( function(sum,n){
	return sum + n 
},0 ) 
```

## 数组方法总结回顾：

1. sort()
- a.sort() 直接用，不放函数，一般就是快排。
- a.sort( function(x , y ){
	return x - y  // 或 y - x 
} )   // 函数里接收两个参数，两个参数的操作决定顺序。

2. a.concat(b)
- concat 一个用处是，连接两个数组，生成新数组。
- 另一用处是，复制数组。复制出来的数组也是新数组，和原数组不相等。

3. a.map()
- 括号里接收一个函数，函数接受两个参数 value key（叫其他名字也行），map 会把函数里的操作收集起来返回出去。forEach 不会这样，这就是两者的区别。

4. a.filter()
- 跟 map 很像，把符合条件的东西过滤出来。接收一个函数，函数里接收两个参数 value key ，也会把结果返回出去。

5. reduce 
- reduce 是以上方法的鼻祖，以上方法是 reduce 的特殊情况。
- reduce 接收一个函数和一个初始值。函数里接收两个参数，比如求和，函数里的参数就是 sum n 。函数里写 reruen sum + n ，sum 意思是上一次的计算结果。整个函数内容意思是，把上一次的计算结果跟下一个数值相加，加完之后，就是整个数组的数值总和了。
